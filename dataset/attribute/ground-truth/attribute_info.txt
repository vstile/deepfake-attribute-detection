This file defines attributes for analyzing video content for my PhD research project. It includes boolean attributes (e.g., is_real, is_alone) and categorical descriptors (e.g., gender, expression, ethnicity) to characterize subjects, movements, and visual features in videos.

info
------------------------------------------------------------------------------

The is_real attribute is set to TRUE if all frames in the video are authentic. If there is any alteration, even for just a few frames, it is considered fake, and the value will be FALSE.

The num_subjects value refers to all people appearing in the scene including the main actor.  The ethnicity is based on the list of ethnic groups used for census by the UK Government:

ASIAN or Asian British
- Indian
- Pakistani
- Bangladeshi
- Chinese
- Any other Asian background

Black, AFRICAN, Caribbean or Black British
- African
- Caribbean
- Any other Black, African or Caribbean background

MIXED or multiple ethnic groups
- White and Black Caribbean
- White and Black African
- White and Asian
- Any other Mixed or multiple ethnic background

WHITE
- English, Welsh, Scottish, Northern Irish or British
- Irish
- Gypsy or Irish Traveller
- Any other White background

OTHER ethnic group
- Arab
- Any other ethnic group

 Ref. https://www.ethnicity-facts-figures.service.gov.uk/style-guide/ethnic-groups/#exceptions, I prefer to use AFRICAN, instead of BLACK to create uniqueness (BLACK is used as hair_color value). 

If hair_lenght is BALD, the hair_color will be OTHER by default.

The is_alone attribute indicates whether there is a single subject in the scene (the primary target actor of manipulations) or if other subjects appear. If another subject appears in the scene, even if only for a few frames, the value will be set to FALSE.

The is_ears_visible attribute indicates if the subject’s ears are visible. If at least one ear appears, even for just a few frames, the value will be TRUE.

In general, all the boolean attributes (except for is_real) are considered TRUE if the condition occurs at any point in the video, even if only for a few frames.
